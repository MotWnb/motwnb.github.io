Welcome to Easy Craft Launcher！
Easy Craft Launcher是一款由中国人使用Python编写的《我的世界》Java版启动器，致力于为玩家提供快速、稳定的游戏体验。
Easy Craft Launcher is a Minecraft Java Edition launcher crafted with Python by a Chinese developer, aiming to deliver a fast and stable gaming experience to players.
虽然目前我们的启动器仅支持命令行操作，没有图形用户界面（GUI），但我们通过GitHub Action实现了自动打包成可执行文件（exe），让玩家轻松安装和启动游戏。
Although our launcher currently operates through the command line without a graphical user interface (GUI), we have implemented a GitHub Action to automatically package it into an executable file (exe), making it easy for players to install and launch the game.
我们的亮点：
Our highlights:
- 一键启动：无需复杂的配置，只需一个命令，即可快速进入游戏。
- One-click Launch: No complex configurations needed - just one command to quickly dive into the game.
- 自动更新：通过GitHub Action，我们确保启动器的版本始终保持最新。
- Automatic Updates: With GitHub Action, we ensure that the launcher is always up-to-date.

- Lightweight: The absence of GUI means fewer resource requirements, allowing for a smoother gaming experience.
- 社区驱动：我们倾听社区的声音，不断优化和改进启动器。
- Community-driven: We listen to the community and continuously optimize and improve the launcher.
